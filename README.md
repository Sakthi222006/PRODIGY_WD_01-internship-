# PRODIGY_WD_01 – Responsive Landing Page

This project is a responsive landing page developed as part of the Prodigy Infotech Web Development Internship. The page features a fixed navigation menu that remains visible across all sections, hover effects on menu items, and dynamic navbar styling that changes when the user scrolls. The design is fully responsive and adapts seamlessly to desktop, tablet, and mobile devices.

## Features
- Fixed Navigation Bar
- Interactive Hover Effects
- Scroll-Based Navbar Color Change
- Responsive Design
- Clean and Modern User Interface

## Technologies Used
- HTML5
- CSS3
- JavaScript

## Outcome
Successfully implemented an interactive and responsive landing page demonstrating front-end development fundamentals and DOM manipulation using JavaScript.
